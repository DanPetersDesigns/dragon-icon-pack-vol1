## Description

The DPD Dragon Icon Pack is a curated fantasy desktop icon collection with transparent PNG dragon icons organized into themed visual series. Styles include alchemy labs, Celtic knotwork, crystal dragons, cute helper dragons, cyber dragons, elemental variants, engraved and etched old-fantasy styles, heraldic crests, Japanese ink dragons, nautical map dragons, Nordic rune dragons, ornate decorative dragons, scholarly dragons, stained glass dragons, steampunk mechanical dragons, tarot-inspired dragons, and stone/gargoyle dragons.

## Release note

The goth and ethereal dragon sets were removed from this release candidate because they were too light or thin in places after transparency cleanup.
