## Summary

This release contains a curated set of transparent PNG fantasy dragon desktop icons organized by themed visual series.

The pack includes alchemy, Celtic, crystal, cute helper, cyber, elemental, engraved, etched, heraldic, Japanese, nautical, Nordic, ornate, scholarly, stained glass, steampunk, tarot, and stone/gargoyle dragon styles.

## Important cleanup note

The goth and ethereal dragon sets were removed from this release candidate because they were too light or thin in places after transparency cleanup.

## Intended use

- Desktop icons
- Launcher shortcuts
- Folder art
- Custom theme packs
- Personal fantasy UI customization
- Creative project organization

- Transparent PNG icons
- Organized by top-level style folders
